Agentic Platform Engineering

GitHub Copilot-powered Infrastructure Automation

What is Git-Ape?

Git-Ape is a platform engineering framework built on GitHub Copilot. It provides a structured, multi-agent system for planning, validating, and deploying Azure infrastructure with security gates, cost analysis, and CI/CD pipeline integration built in.

The Deployment Flow

1
Collect Inputs — Requirements Gatherer interviews the user, validates CAF naming and SKU availability
2
Generate & Analyze — Template Generator produces ARM template, architecture diagram, cost estimate, and security report
3
Security Gate — Blocking check; failures loop back for fixes
4
WAF Review — Principal Architect scores against the 5 Well-Architected Framework pillars
5
User Confirmation — Nothing deploys without explicit approval
6
Deploy — Resource Deployer runs az deployment, monitors, and retries
7
Validate — Health checks, endpoint tests, architecture diagram generation

Core Agents

@git-ape

Main orchestrator; coordinates stages, enforces security gates, delegates to subagents

Requirements Gatherer

Interviews user, validates CAF naming, checks SKU availability

Template Generator

Creates ARM templates, architecture diagrams, cost estimates

Resource Deployer

Runs az deployment, monitoring, retry, integration tests

Principal Architect

WAF 5-pillar review, trade-off analysis

IaC Exporter

Imports live resources into templates

Execution Modes

Interactive

Talk to @git-ape in VS Code Copilot Chat, authenticate via az login, approve each step in real time.

Headless

Copilot Coding Agent picks up a GitHub issue, generates templates on a branch, opens a PR, and CI/CD workflows handle validation, deployment, and teardown via OIDC.

CI/CD Workflows

  • git-ape-plan.yml — PR with template changes → validate, what-if, post plan as PR comment
  • git-ape-deploy.yml — Merge to main or /deploy comment → execute ARM deployment
  • git-ape-destroy.yml — Merge PR with destroy-requested → delete resource group
  • git-ape-verify.yml — Manual dispatch → verify OIDC, RBAC, pipeline health

Get Started

Install:

  • copilot plugin marketplace add Azure/git-ape
  • copilot plugin install Azure/git-ape

Example Prompts:

  • @git-ape deploy a Python function app
  • @git-ape deploy a web app with SQL database
  • @Git-Ape Onboarding set up this repo for Azure deployments
  • @git-ape destroy Python function app
