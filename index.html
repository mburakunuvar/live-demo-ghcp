GitHub Copilot
Across Surfaces

GitHub Copilot is no longer just an IDE autocomplete tool — it's an AI system that spans every surface where developers work. The same agent loop, the same customization primitives, and the same agentic capabilities are available across four surfaces: GitHub.com, the IDE, the CLI, and the SDK.

🌐

GitHub.com

Copilot is embedded directly in the GitHub platform — issues, PRs, code review, and the Copilot Coding Agent.

💻

IDE

Chat, inline completions, multi-file edits, and agent mode in VS Code, JetBrains, Visual Studio, and more.

⌨️

CLI

Natural-language commands in the terminal. Explain, suggest, and run shell commands powered by AI.

🔧

SDK

Build custom Copilot extensions and agents. Integrate Copilot capabilities into your own tools and workflows.
